# Projet Titanic Survival Prediction - Ingénierie Logicielle

## Description du projet

Ce projet applique les bonnes pratiques d'ingénierie logicielle à un projet de Machine Learning existant : la prédiction de survie sur le Titanic.

L'objectif est de refactoriser le code d'un notebook Jupyter en un projet Python professionnel avec :
- Scripts Python modulaires et réutilisables
- Tests unitaires
- Documentation complète
- Pipeline CI/CD
- Gestion de version avec Git

## Contexte académique

**Établissement :** IUT Paris Cité - BUT VCOD

## Source des données

Les données proviennent du challenge Kaggle : [Titanic - Machine Learning from Disaster](https://www.kaggle.com/c/titanic)
